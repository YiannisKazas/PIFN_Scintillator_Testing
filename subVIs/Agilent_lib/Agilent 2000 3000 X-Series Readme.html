3. Known Issues

  • Configure Arbitrary Waveform Data.vi
    • While using commands in this VI(WGEN:ARB:DATA and WGEN:ARB:DATA:DAC) as Programmer's Guide described, error "-222, Data out of range" is reported. These two commands are swapped in the VI to avoid error report, but the functionality are not fully tested yet.

    To report issues or provide feedback about this instrument driver, please send an email to instrument.drivers@ni.com.
